💰 SMART BUDGETING WEB APP FOR STUDENTS
A simple and user-friendly web application designed to help students manage their finances efficiently. This app allows users to track income and expenses, set budgets, and monitor savings in an organized way.

Logout functionality

⚙️ How It Works
User logs in using valid credentials
Dashboard loads by default
User can add transactions, view or search transactions, and set budget
Data is stored in the database
Dashboard updates automatically
Alerts notify the user about spending limits
▶️ How to Run
Download or clone this repository
Open the project folder
Run the index.html file in your browser
Login and start using the application
✨ Conclusion
The SMART BUDGETING WEB APP FOR STUDENTS provides a simple and effective way for students to manage their finances. It helps users track spending, stay within budget, and make better financial decisions.
